# RigService

RigService provides static utility methods for working with armatures, rigs, bones, bone constraints, poses, and weights. It handles rig creation, bone manipulation, weight management, pose import/export, Rigify integration, and armature refitting to new basemeshes.

## Source

`src/mpfb/services/rigservice.py`

## Dependencies

- `LogService` — logging
- `LocationService` — path resolution for poses and weights
- `SystemService` — system utilities
- `TargetService` — shape key operations for refitting
- `ObjectService` — Blender object operations
- `SkeletonObjectProperties` — skeleton custom property access

## Public API

### ensure_global_poses_are_available()

Copy pose files from the MPFB source data to the user directory.

### apply_pose_as_rest_pose(armature_object)

Apply the current pose as the rest pose for the armature.

### ensure_armature_modifier(obj, armature_object, *, move_to_top=True, subrig=None)

Ensure the object has an armature modifier pointing to the given armature.

### get_world_space_location_of_pose_bone(bone_name, armature_object)

Find the head and tail locations of a bone in world space.

### find_pose_bone_by_name(name, armature_object)

Find a pose bone by name.

### find_edit_bone_by_name(name, armature_object)

Find an edit bone by name (requires edit mode).

### activate_pose_bone_by_name(name, armature_object, also_select_bone=True, also_deselect_all_other_bones=True)

Activate a specific bone in pose mode.

### remove_all_constraints_from_pose_bone(bone_name, armature_object)

Remove all constraints from the specified pose bone.

### add_bone_constraint_to_pose_bone(bone_name, armature_object, constraint_name)

Add a constraint to a pose bone.

### add_copy_rotation_constraint_to_pose_bone(...)

Add a Copy Rotation constraint to a pose bone.

### add_rotation_constraint_to_pose_bone(...)

Add a Limit Rotation constraint to a pose bone.

### add_ik_rotation_lock_to_pose_bone(...)

Add an IK rotation lock to a pose bone.

### add_ik_constraint_to_pose_bone(...)

Add an IK constraint to a pose bone.

### find_pose_bone_tail_world_location(bone_name, armature_object)

Find the world-space location of a bone's tail.

### find_pose_bone_head_world_location(bone_name, armature_object)

Find the world-space location of a bone's head.

### set_ik_rotation_limits(bone_name, armature_object, axis, min_angle=0, max_angle=0)

Set IK rotation limits for a bone on the specified axis.

### get_bone_orientation_info_as_dict(armature_object)

Retrieve orientation info of all bones as a dictionary.

### set_bone_orientation_from_info_in_dict(armature_object, new_bone_orientation)

Set bone orientations from a dictionary.

### create_rig_from_skeleton_info(name, data, parent=None, scale=0.1)

Create a rig from skeleton info data.

### ensure_armature_has_bone_shape_objects_as_children(armature_object)

Ensure the armature has bone shape objects as children.

### display_pose_bone_as_empty(armature_object, bone_name, empty_type="SPHERE", scale=1.0)

Display a bone as an empty object of the specified type.

### get_weights(armature_objects, basemesh, exclude_weights_below=0.0001, all_groups=False, all_bones=False, all_masks=True)

Create an MHW-compatible weights dictionary from the current rig setup.

### load_weights(armature_objects, basemesh, mhw_filename, *, all=False, replace=False)

Load weights from a JSON file and apply them.

### apply_weights(armature_objects, basemesh, mhw_dict, *, all=False, replace=False)

Apply weights from a dictionary to the basemesh.

### set_extra_bones(armature_object, extra_bones)

Set extra bones metadata on the armature.

### get_extra_bones(armature_object)

Get the list of extra bones metadata from the armature.

### get_deform_group_bones(armature_object)

Get the list of deform group bones from the armature.

### find_extra_bones(armature_object)

List deform bones generated by Rigify.

### identify_rig(armature_object)

Identify the rig type of an armature.

### get_rig_weight_fallbacks(rig_type)

Get the list of rig types to try when loading weights, in fallback order.

### mirror_bone_weights_to_other_side_bone(...)

Mirror bone weights from one side to the corresponding bone on the other side.

### mirror_bone_weights_inside_center_bone(...)

Mirror weights within a center bone.

### symmetrize_all_bone_weights(armature_object, left_to_right=False, rig_type=None)

Symmetrize all bone weights across the rig.

### set_pose_from_dict(armature_object, pose, from_rest_pose=True)

Set a pose from a dictionary of bone rotations.

### get_pose_as_dict(armature_object, root_bone_translation=True, ik_bone_translation=True, fk_bone_translation=False, onlyselected=False)

Get the current pose as a dictionary.

### refit_existing_armature(armature_object, basemesh)

Refit an armature to a new basemesh shape.

### refit_existing_subrig(armature_object, parent_rig)

Refit a subrig to a new parent rig.

### normalize_rotation_mode(armature_object, rotation_mode="XYZ")

Normalize the rotation mode of all bones.

### find_leg_length(armature_object)

Find the total length of the leg bones.

### find_arm_length(armature_object)

Find the total length of the arm bones.

### copy_pose(from_armature, to_armature, only_rotation=True)

Copy a pose from one armature to another.

### add_path_object_to_bone(armature_object, bone_name, bevel_depth=0.015)

Create a NURBS path object attached to a bone.

### add_uv_sphere_object_to_bone(armature_object, bone_name, sphere_scale=0.01, tail_rather_than_head=False)

Create a UV sphere object attached to a bone's head or tail.

## Example

```python
from mpfb.services.rigservice import RigService

RigService.ensure_global_poses_are_available()
pose = RigService.get_pose_as_dict(armature_object)
RigService.set_pose_from_dict(armature_object, pose)
rig_type = RigService.identify_rig(armature_object)
```
